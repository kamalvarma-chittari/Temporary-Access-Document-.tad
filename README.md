# TAD (Temporary Access Document)

> **Rethinking document ownership beyond traditional file sharing.**

## Overview

TAD (Temporary Access Document) is a proof-of-concept project that explores a different approach to digital document protection by enabling document owners to define how their documents can be accessed and used.

Unlike conventional file sharing, where control is largely lost once a document is distributed, TAD demonstrates the concept of owner-controlled access through configurable document usage policies.

The current repository contains a web-based prototype developed to validate the concept and demonstrate the overall workflow.

## Prototype Features

* TAD Maker
* TAD Viewer
* Temporary document access
* Document expiry
* View restrictions
* Print control
* Browser-based proof of concept

## Project Status

This project is currently in the prototype stage and has been developed to validate the core concept and workflow. It is intended for research, experimentation, and further development.

## Vision

The long-term objective is to evolve TAD into secure desktop and mobile applications that operate entirely offline, without requiring users to upload documents to any server. Document processing and protection are envisioned to take place locally on the user's device while supporting multiple document formats such as PDF, DOC/DOCX, and images through a unified owner-controlled document platform.

## Repository

This repository represents the initial prototype of the TAD concept and is shared for demonstration and learning purposes.
